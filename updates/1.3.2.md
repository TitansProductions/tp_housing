**TP Housing**

# What has been updated?

## Changelog - (Added, Fixed, Removed):

1. Added the possibility when a location is about to set (wardrobe, storage, etc), to also have an extra option to cancel action.
2. Added in the config (2) new options that allow you to open storage or wardrobe through the menu than having locations with prompts. This feature is great for small properties or for the ones who don't really want a property to have different positions for every actions.
3. Added voice-talk support for the new UI that replaces `tpz_menu_base` in order to support voice chat while NUI is active.
4. Added an extra distance check to prevent wardrobe, storage and other positions to prevent them from being set near the menu prompt.
5. Added possibility to pay the taxes through TP Banks (Our paid script).
6. Added extra permissions for keyholder members.
7. Added on upgrades a new feature of increasing the default keyholder members by upgrading a property - totally optional.
8. Added a new database table column called `materials` which is going to contain all the materials you are going to use for upgrades.

4. Add the new thread on `client/tp-client_escrow_allow.lua` to block key actions when having NUI active - this is required for those who have `Config.PushToTalkFunction` active.

```lua
-----------------------------------------------------------
--[[ Threads ]]--
-----------------------------------------------------------

if Config.PushToTalkFunction then
    -- The following task is made to prevent opening specific keys while nui is active.
    Citizen.CreateThread(function()
        while true do 
            
            local sleep  = 1500
        
            local player = PlayerPedId()
            local isDead = IsEntityDead(player)
    
            if not GetPlayerData().Loaded then
                goto END
            end
    
            if GetPlayerData().IsInMenu then 
                sleep = 0
    
                DisableControlAction(0, Config.HorseWhistleKey, true) -- Horse Whistle
                DisableControlAction(0, Config.WagonWhistleKey, true) -- Wagon Whistle
                
                DisableControlAction(0, 0xE6F612E4, true) -- 1
                DisableControlAction(0, 0x1CE6D9EB, true) -- 2
                DisableControlAction(0, 0x4F49CC4C, true) -- 3
                DisableControlAction(0, 0x8F9F9E58, true) -- 4
                DisableControlAction(0, 0xAB62E997, true) -- 5
                DisableControlAction(0, 0xA1FDE2A6, true) -- 6
                DisableControlAction(0, 0xB03A913B, true) -- 7
                DisableControlAction(0, 0x42385422, true) -- 8
    
                -- We close the NUI if player is dead.
                if isDead then 
                    CloseNUIProperly()
                end
    
            end
    
            ::END:: 
            Wait(sleep)
    
        end
    
    end)
end
```

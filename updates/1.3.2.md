**TP Housing**

# What has been updated?

## Changelog - (Added, Fixed, Removed):

1. Added the possibility when a location is about to set (wardrobe, storage, etc), to also have an extra option to cancel action.
2. Added in the config (2) new options that allow you to open storage or wardrobe through the menu than having locations with prompts. This feature is great for small properties or for the ones who don't really want a property to have different positions for every actions.

- client/tp-client_menu.lua
- client/tp-client_nui.lua
- client/tp-client_nui_management.lua (Add new file)
- client/tp-client_main.lua
- html (Replace the whole folder)
- locales.lua (Replace to latest - the changes are tons!)
- fxmanifest.lua

## Notes:

1. Add the (2) new configuration options on `config.lua` file related to storage and wardrobe menu access.

```lua
-- Set to false if you want the wardrobe access to be through a location instead through the menu.
Config.DisableWardobePositionSet = true -- 1.3.2

-- Set to false if you want the storage access to be through a location instead through the menu.
Config.DisableStoragePositionSet = true -- 1.3.2
```

2. Replace `Config.ManagementMenu` to the new one for supporting the UI that replaced the tpz_menu_base script.

```lua

-- FROM

-- Options in the property management menu to display / not based on your personal and server preferences.
-- (!) DO NOT MODIFY THE TYPE NAMES.
Config.ManagementMenu = {
    { Type = 'MENU_UPGRADES',          Enabled = true  }, -- upgrades menu option
    { Type = 'MENU_FURNITURE',         Enabled = true}, -- furniture placement and overview menu option
    { Type = 'MENU_WARDROBE_LOCATION', Enabled = true}, -- set wardrobe location menu option
    { Type = 'MENU_STORAGE_LOCATION',  Enabled = true}, -- set storage location menu option
    { Type = 'MENU_LEDGER',            Enabled = true }, -- deposit or withdraw money from the ledger menu option
    { Type = 'MENU_SET_KEYHOLDERS',    Enabled = true}, -- set property keyholders (members to have access to the house) menu option
    { Type = 'MENU_TRANSFER',          Enabled = true}, -- transfer the property to another player menu option
    { Type = 'MENU_SELL',              Enabled = true}, -- sell the property menu option
}

-- TO

-- Options in the property management menu to display / not based on your personal and server preferences.
-- (!) DO NOT MODIFY THE TYPE NAMES.
-- @param BackgroundImage : Images located at html/img/management folder (version 1.3.2)
Config.ManagementMenu = {
    { Type = 'MENU_UPGRADES',          Enabled = true, BackgroundImage = 'upgrades.jpg' }, -- upgrades menu option
    { Type = 'MENU_FURNITURE',         Enabled = true, BackgroundImage = 'furnitures.jpg'  }, -- furniture placement and overview menu option
    { Type = 'MENU_WARDROBE_LOCATION', Enabled = true, BackgroundImage = 'wardrobe.jpg'  }, -- set wardrobe location menu option
    { Type = 'MENU_STORAGE_LOCATION',  Enabled = true, BackgroundImage = 'storage.jpg'  }, -- set storage location menu option
    { Type = 'MENU_LEDGER',            Enabled = true, BackgroundImage = 'ledger.jpg'  }, -- deposit or withdraw money from the ledger menu option
    { Type = 'MENU_SET_KEYHOLDERS',    Enabled = true, BackgroundImage = 'keyholders.jpg'  }, -- set property keyholders (members to have access to the house) menu option
    { Type = 'MENU_TRANSFER',          Enabled = true, BackgroundImage = 'transfer.jpg'  }, -- transfer the property to another player menu option
    { Type = 'MENU_SELL',              Enabled = true, BackgroundImage = 'sell.jpg' }, -- sell the property menu option
}
```

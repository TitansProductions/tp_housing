# What has been updated?

## Changelog - (Added, Fixed, Removed):

1. Added new design for the keyholders management to a more advanced one and also more understandable.

2. Added new design for the upgrades management with better display of the required materials, cost and the extra given storage.

3. Added a new text display on set wardrobe, storage, etc. The previous text display was more bold with a different font which was not great.

4. Fixed an issue when stopping / restarting the script while the player was in the house management menu, it would freeze the player screen and the only possible way would be to restart `tpz_menu_base` or a relog.

5. Added possibility to use `\n` symbol for creating a new line on the required materials where `DisplayRequiredLabel` is displayed on `config_upgrades.lua` - example:

```lua
DisplayRequiredLabel = 'X5 Wooden Nails\nX10 Wood.' --< Used \n for a new line.
```

## How to update?

- You can easily update the modified files by redownloading from the **Cfx.re Portal** and copy-paste on the mentioned directory file paths.
- If new options have been implemented to the configuration file or edited, **DO NOT** copy-paste the new configuration file, otherwise you will get the latest configuration but you will lose all the changes you have made. The proper way updating a configuration file is by finding the new configuration options or the modified ones and do it **manually**.

*Cfx.re Portal **DOES NOT** force-update your configuration file or any files, if it did, you would always lose all the configuration changes you made or would totally break the script, all files are updated or edited manually.*
